# config/

Ce dossier contient les fichiers de configuration du projet **NEMO-PROJECT**.

## Contenu
- `nemo.conf` — configuration générale du projet  
- `preferences.conf` — préférences utilisateur et options activables  
- `modules.conf` — activation/désactivation des modules  
- `config-loader.sh` — chargeur centralisé utilisé par les scripts

## Objectif
Centraliser toutes les configurations du projet pour garantir une structure propre, modulaire et facilement extensible.

Les scripts du projet chargent automatiquement ces fichiers.
