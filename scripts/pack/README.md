 # packs/

Ce dossier contient les packs avancés du projet **NEMO-PROJECT**.  
Chaque pack est un script Bash autonome permettant d’installer un ensemble de fonctionnalités.

## Contenu
- `security-pack.sh` — outils et modules de sécurité  
- `multimedia-pack.sh` — outils audio/vidéo et conversions  
- `dev-pack.sh` — environnement développeur  
- `network-pack.sh` — outils réseau  
- `ultimate-pack.sh` — installe tous les packs

## Objectif
Fournir une installation modulaire, propre et rapide des fonctionnalités avancées du projet.

Les packs modifient automatiquement les modules dans `config/modules.conf`.
