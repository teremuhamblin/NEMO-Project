# Nemo-Project 
>Projet modulaire pour améliorer, optimiser et étendre le gestionnaire de fichiers **Nemo** sous Linux Mint Cinnamon.

## Objectifs
- Ajouter des actions Nemo personnalisées  
- Améliorer l’ergonomie et la productivité  
- Fournir des scripts propres, robustes et modulaires  
- Permettre une installation simple et réversible  

## Structure
```text
actions/      → Actions Nemo + scripts associés
config/       → Fichiers de configuration Nemo
scripts/      → Installateur, désinstallateur, utilitaires
docs/         → Documentation du projet
```

## Installation
```text
sudo bash scripts/install.sh
```

## Désinstallation
```text
sudo bash scripts/uninstall.sh
```
